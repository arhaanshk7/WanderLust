🏡 WanderLust – Hospitality & Travel Accommodation Platform

📌 Overview

StayNest is a full-stack Hospitality & Travel Accommodation Platform inspired by modern booking applications. It enables travelers to discover and book accommodations while allowing hosts to list and manage their properties through a secure and user-friendly interface.

The project focuses on building a real-world web application with authentication, booking management, real-time communication, cloud image storage, and scalable backend architecture.

---

✨ Features

- 🔐 Secure User Authentication & Authorization
- 🏠 Create, Update & Delete Property Listings
- 📷 Cloud Image Uploads with Cloudinary
- 📅 Property Booking System
- ⭐ Ratings & Reviews
- 💬 Real-Time Chat using Socket.IO
- 👤 User Profile & Booking Management
- 🛡️ Role-Based Access Control
- 📱 Responsive User Interface
- ⚠️ Robust Error Handling & Validation
